# shaders/

The canonical PS1 shader ships **inside the addon** at
`addon/patina/ps1.gdshader` so the addon is self-contained when you copy it to
`res://addons/patina/`. The copy here is the same file, kept at the TDD's
documented `shaders/` location for reference. If you edit one, edit both — or
delete this copy and treat the in-addon one as the single source.
